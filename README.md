## Parametrização de Formulários (Python + JSON)

Projeto prático desenvolvido para estudar a leitura, alteração e validação
de formulários em JSON utilizando Python.

O notebook simula um cenário comum de implantação de sistemas, onde um
formulário base é carregado, recebe novos campos com ID automático e gera
um arquivo final validado.

### Arquivos do projeto
- `parametrizacao_formularios.ipynb` → notebook com o passo a passo e validações
- `formulario_base.json` → formulário original
- `formulario_final.json` → formulário gerado após a parametrização
